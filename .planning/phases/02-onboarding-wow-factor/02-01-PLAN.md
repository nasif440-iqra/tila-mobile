    - `opacity?: number` (default 0.12) — used only when `animated=false`
    - `animated?: boolean` (default false) — backward compatible, existing callers see no change
    - `color?: string` (optional) — when provided, use this color at full opacity in the background and control opacity via animated style. When not provided, use `rgba(196, 164, 100, <opacity>)` as before.
    - `pulseMin?: number` (default 0.08)

    When `animated=true`: render an `Animated.View` from react-native-reanimated. Use `useSharedValue` starting at `pulseMin`. In a `useEffect`, set `pulseOpacity.value = withRepeat(withSequence(withTiming(pulseMax, { duration: 2000, easing: Easing.inOut(Easing.ease) }), withTiming(pulseMin, { duration: 2000, easing: Easing.inOut(Easing.ease) })), -1, false)`. Apply via `useAnimatedStyle(() => ({ opacity: pulseOpacity.value }))`. The background color should be the color prop at full opacity (or `rgba(196, 164, 100, 1)` if no color prop), with opacity controlled entirely by the animated style.
