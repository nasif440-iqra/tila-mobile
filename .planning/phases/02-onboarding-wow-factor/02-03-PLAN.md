    Create `src/components/shared/BismillahOverlay.tsx`. This is the per-session Bismillah overlay for lesson entry.

    **Session detection (module-level, per RESEARCH recommendation):**
    At the top of the file, outside any component:
    ```typescript
    let bismillahShownThisSession = false;

    export function shouldShowBismillah(): boolean {
      return !bismillahShownThisSession;
    }

    export function markBismillahShown(): void {
      bismillahShownThisSession = true;
    }
    ```
    This resets when the app process is killed (new session) and persists when backgrounded (same session). Simpler and more correct than SecureStore timestamps.

    **BismillahOverlay component:**
    Props: `{ onComplete: () => void }`

    1. On mount: call `markBismillahShown()` immediately, call `hapticSelection()`.
    2. Set up auto-dismiss: after 2500ms, trigger fade-out. After fade-out (500ms), call `onComplete`.
    3. Use `useSharedValue(1)` for overlay opacity. After 2500ms timeout, animate to 0 with `withTiming(0, { duration: 500 })`. Use `runOnJS` callback or a separate useEffect watching a `dismissing` state to call `onComplete` after fade.

    Implementation approach for dismiss timing:
    ```typescript
    const [dismissing, setDismissing] = useState(false);
    const overlayOpacity = useSharedValue(1);

    useEffect(() => {
      markBismillahShown();
      hapticSelection();
      const timer = setTimeout(() => {
        setDismissing(true);
        overlayOpacity.value = withTiming(0, { duration: 500 });
      }, 2500);
      return () => clearTimeout(timer);
    }, []);

    useEffect(() => {
      if (dismissing) {
        const fadeTimer = setTimeout(onComplete, 550); // slightly after 500ms fade
        return () => clearTimeout(fadeTimer);
      }
    }, [dismissing, onComplete]);
    ```

    4. Render as absolutely positioned full-screen overlay:
