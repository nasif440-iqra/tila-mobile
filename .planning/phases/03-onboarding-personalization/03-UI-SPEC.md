## Spacing Scale

Declared values from `src/design/tokens.ts` (8px base rhythm):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, divider margins |
| md | 12px | Intra-group spacing (Arabic-to-transliteration gap) |
| lg | 16px | Section padding, inter-element spacing |
| xl | 24px | Section breaks, vertical rhythm between content groups |
| xxl | 32px | Layout gaps, major content separation |
| xxxl | 48px | Page-level breathing room |
| xxxxl | 64px | Top/bottom page margins |

Exceptions:
- PhraseReveal vertical word units use `xl` (24px) between units in vertical layout for meditative spacing
- PhraseReveal Arabic-to-transliteration gap uses `xs` (4px) -- tight pairing to show connection
- PhraseReveal transliteration-to-meaning gap uses `sm` (8px) -- slightly more separation for hierarchy
- Finish screen checkmark-to-headline gap uses `xl` (24px) -- matches current implementation

Source: `src/design/tokens.ts` spacing object (pre-existing)

## Animation Contract

### PhraseReveal Timing

| Parameter | Value | Source |
|-----------|-------|--------|
| Per-word fade duration | 700ms (default, range 600-800ms) | SACR-01 requirement, D-02 decision |
| Stagger between words | 350ms (default, range 300-400ms) | SACR-01 requirement, D-02 decision |
| Arabic opacity easing | `Easing.out(Easing.cubic)` | Matches `easings.contentReveal` from animations.ts |
| Transliteration delay | +200ms after Arabic starts | Claude's discretion -- tight follow for connection |
| Transliteration duration | 80% of wordDuration (560ms) | Claude's discretion -- slightly faster, secondary |
| Meaning delay (Bismillah only) | +400ms after Arabic starts | Claude's discretion -- third in sequence |
| Meaning duration | 70% of wordDuration (490ms) | Claude's discretion -- fastest, tertiary |
| Skip behavior | Tap anywhere shows all words instantly | D-02 decision |
| onComplete fires | After last word fully revealed OR immediately on skip | Implementation requirement |

### Bismillah Total Reveal Duration
- 4 words: (3 x 350ms stagger) + 700ms word + 400ms meaning offset = ~2,150ms total
- Plus CTA fade-in after: 400ms
- Total time to interactive: ~2,550ms (was 4,000ms auto-advance)

### Finish Screen Settle

| Parameter | Value | Source |
|-----------|-------|--------|
| Initial scale | 0.98 (was 0.5 with bouncy spring) | D-13 decision |
| Initial opacity | 0 | D-13 decision |
| Fade-in duration | 600ms | Claude's discretion |
| Scale up | 0.98 to 1.02, 600ms, `Easing.out(Easing.cubic)` | D-13 decision |
| Scale settle | 1.02 to 1.0, 400ms, `Easing.inOut(Easing.ease)` | D-13 decision |
| Total settle | 1,000ms | Derived from above |

### Reduce Motion Fallback

| Component | Reduced Motion Behavior | Source |
|-----------|------------------------|--------|
| PhraseReveal | All words visible immediately, single 300ms opacity fade | Claude's discretion, matches WarmGlow/FloatingLetters pattern |
| Finish checkmark | scale=1, opacity=1, no animation, hapticSuccess still fires | FOUN-04 requirement |

Source: `src/design/animations.ts`, CONTEXT.md decisions, REQUIREMENTS.md SACR-01

## Screen-by-Screen Visual Contract

### Bismillah Screen (SACR-02)

**Layout:** `OnboardingStepLayout variant="splash"` (centered, full-height)
**Background:** AtmosphereBackground 'onboarding' (from flow wrapper)
**Content structure (top to bottom, centered):**
1. PhraseReveal (vertical layout) -- 4 word units stacked:
   - Each unit: Arabic (Amiri 36px, color `primaryDark`) / transliteration (Inter 15px, color `textMuted`) / meaning (Lora 13px italic, color `textSoft`)
   - `xl` (24px) vertical gap between units
   - `xs` (4px) gap Arabic-to-transliteration
   - `sm` (8px) gap transliteration-to-meaning
2. Footer: "Continue" Button (appears after reveal completes with 400ms FadeIn)
**Audio:** `playSacredMoment()` on mount
**Haptic:** `hapticSelection()` on mount
**Removed:** 4-second auto-advance timer, single-line Bismillah display, italic English translation

### Tilawah Screen (SACR-04)

**Layout:** `OnboardingStepLayout variant="splash"`
**Background:** AtmosphereBackground 'onboarding' (from flow wrapper)
**Content structure (top to bottom, centered):**
1. PhraseReveal (single word): Arabic `تِلاوَة` (Amiri 56px, custom override, color `primaryDark`) / transliteration "Tilawah" (Inter 15px, color `textMuted`)
2. `lg` (16px) spacer
3. Headline: "To recite the Quran beautifully is Tilawah" -- "Tilawah" rendered in Lora Italic 26px, accent color (static, no shimmer)
4. `md` (12px) spacer
5. Motto: "Recite. Reflect. Return." (Inter 12px, letter-spacing 1.8, color `textMuted`)
6. Footer: "Begin" Button
**Removed:** ShimmerWord component (withRepeat opacity animation)

### Hadith Screen (SACR-05)

**Layout:** `OnboardingStepLayout variant="splash"`
**Background:** AtmosphereBackground 'onboarding' (from flow wrapper)
**Atmosphere layers (behind content):**
- WarmGlow (size 340, animated, pulseMin 0.08, pulseMax 0.18) -- existing
- ArchOutline (accent color, 12% opacity) -- existing
**Content structure (top to bottom, centered):**
1. Headline: "Struggling is not failing" (Lora 28px SemiBold Italic, color `brown`) -- existing stagger entrance
2. Gold diamond separator (6x6px, 45deg rotation, 60% opacity, color `accent`) -- existing
3. `lg` (16px) spacer
4. PhraseReveal (horizontal layout): Arabic hadith words with transliteration beneath each
5. `lg` (16px) spacer
6. English translation: "The one who struggles with the Qur'an receives a double reward." -- appears after PhraseReveal completes (400ms FadeIn, Lora 17px italic, color `textSoft`)
7. `lg` (16px) spacer
8. Divider line (28px wide, 1px, accent at 40% opacity) -- existing
9. Source: "SAHIH AL-BUKHARI 4937" -- existing
10. Footer: "Continue Journey" Button

### Finish Screen (SACR-06)

**Layout:** `OnboardingStepLayout variant="splash"`
**Background:** AtmosphereBackground 'onboarding' (from flow wrapper)
**Content structure (top to bottom, centered):**
1. Ambient Alif watermark (200px, 8% opacity, absolute positioned) -- existing, D-14
2. Checkmark circle (72px, `accentLight` bg, accent border) -- gentle scale settle (0.98 to 1.02 to 1.0) + fade-in, NOT bouncy spring
3. `xl` (24px) spacer
4. Headline: "You've already begun" (Lora 28px Bold, color `brown`, letter-spacing -0.5)
5. `md` (12px) spacer
6. Body: "Now let's take your first real lesson." (Inter 15px, color `textSoft`, max-width 300)
7. Footer: "Start Lesson 1" Button (or "Try Again" on error)

### Welcome Screen (SACR-03)

**No content changes.** AtmosphereBackground applied at OnboardingFlow level provides the atmosphere upgrade. Existing BrandedLogo and stagger animations remain.
