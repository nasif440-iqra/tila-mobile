**Types (exported):**
```typescript
export interface PhraseWord {
  arabic: string;
  transliteration: string;
  meaning?: string; // Only for Bismillah per D-04
}

export interface PhraseRevealProps {
  words: PhraseWord[];
  wordDuration?: number;   // default 700, range 600-800ms per D-02
  staggerDelay?: number;   // default 350, range 300-400ms per D-02
  onComplete?: () => void; // fires after all words revealed or on skip
  layout?: 'horizontal' | 'vertical'; // vertical for Bismillah (D-08), horizontal for others
  arabicSize?: 'large' | 'display'; // default 'large' (36px), allow override for Tilawah 56px
  arabicStyle?: object; // style override for Arabic text (Tilawah needs custom fontSize)
  accessibilityLabel?: string;
}
```

**RevealWord internal component** (not exported -- each instance owns its own shared values to comply with React hook rules):
- Props: `word: PhraseWord`, `index: number`, `staggerDelay: number`, `wordDuration: number`, `skipImmediate: boolean`
- Uses `useSharedValue` for `arabicOpacity`, `translitOpacity`, `meaningOpacity` (all start at `skipImmediate ? 1 : 0`)
- On mount (useEffect), if not skipImmediate:
  - `arabicOpacity.value = withDelay(index * staggerDelay, withTiming(1, { duration: wordDuration, easing: Easing.out(Easing.cubic) }))`
  - `translitOpacity.value = withDelay(index * staggerDelay + 200, withTiming(1, { duration: wordDuration * 0.8, easing: Easing.out(Easing.cubic) }))`
  - If `word.meaning`: `meaningOpacity.value = withDelay(index * staggerDelay + 400, withTiming(1, { duration: wordDuration * 0.7, easing: Easing.out(Easing.cubic) }))`
- When `skipImmediate` changes from false to true, set all opacity values to `withTiming(1, { duration: 150 })` for instant reveal
- Renders a View container with:
  - Animated ArabicText (using the parent's arabicSize prop, color `primaryDark` from useColors)
  - Animated Text for transliteration (Inter 15px, color `textMuted`, marginTop: spacing.xs (4px))
  - Conditional Animated Text for meaning (Lora Italic 12px, color `textSoft`, marginTop: spacing.sm (8px))

**PhraseReveal parent component:**
- Calls `useReducedMotion()` from Reanimated
- State: `skipped` (boolean, initialized to `reducedMotion ?? false`)
- Timer ref for completion callback
- On mount: if not skipped, calculate total time as `(words.length - 1) * staggerDelay + wordDuration + 400` and set setTimeout for onComplete. If skipped, call onComplete immediately.
- `handleSkip` function: clear timer, set skipped=true, call onComplete
- Wraps children in `Pressable` with `onPress={handleSkip}` and accessibility hint "Double tap to reveal all words"
- Sets `accessibilityLabel` on the Pressable if provided
- Container style: vertical layout uses `{ alignItems: 'center', gap: spacing.xl (24px) }`, horizontal uses `{ flexDirection: 'row', flexWrap: 'wrap', justifyContent: 'center', gap: spacing.lg (16px) }`
- Maps over words, rendering RevealWord for each with `skipImmediate={skipped}`

Add line: `export { PhraseReveal } from "./PhraseReveal";`
Also add type re-export: `export type { PhraseWord, PhraseRevealProps } from "./PhraseReveal";`
