## Spacing Scale

Declared values from `src/design/tokens.ts` (8px base rhythm):

| Token | Value | Usage in this phase |
|-------|-------|---------------------|
| xs | 4px | Icon gaps, pill internal padding, badge gaps |
| sm | 8px | Compact element spacing, label margins |
| md | 12px | Pill padding, current-label inner padding |
| lg | 16px | Section padding, card inner sections |
| xl | 24px | Scroll content padding, section gaps, card vertical padding |
| xxl | 32px | Hero card vertical padding, major section breaks |
| xxxl | 48px | Journey node row spacing (vertical rhythm) |
| xxxxl | 64px | Hero card bottom margin (separation from journey) |

## Visual Specifications

### Header Region

- Layout: Row, space-between, vertically centered
- Left: "tila" in `pageTitle` preset (Lora Italic 24px), color `brown` (#3D2B1F)
- Right: AnimatedStreakBadge (only visible when `currentWird > 0`)
- Bottom margin: `xl` (24px)

### AnimatedStreakBadge

- Container: Pill shape (`borderRadius: 9999`), border 1px `border` color, horizontal padding `md` (12px), vertical padding `xs` (4px)
- Content row: Crescent icon (14px, `accent` color) + count (`bodySemiBold` 13px, `text` color) + "Wird" label (`bodyMedium` 11px, `textMuted` color), gap `xs` (4px)
- Animation: Gentle breathing glow behind the badge using WarmGlow (size: 60px, animated, pulseMin: 0.04, pulseMax: 0.12, positioned centered behind badge, color: `accentGlow`)
- Entrance: FadeIn with duration `normal` (300ms), easing `contentReveal`, delay 200ms (staggered after header text)
- When streak reaches a new day milestone: brief scale pulse using `springs.bouncy` from 1.0 to 1.05 back to 1.0

### Hero Card

- Wrapper: `Card` component with `elevated` prop, `borderRadius: xl` (24px), vertical padding `xxl` (32px)
- Bottom margin: `xxxxl` (64px) to separate from journey section
- Content alignment: center

**Phase pill:**
- Background: `bg` (#F8F6F0), pill shape (borderRadius full), padding `xs`/`md` vertical/horizontal
- Text: `label` preset, color `accent`
- Content: "Phase {N} -- {Phase Name}"

**Letter circle:**
- Size: 112x112px, borderRadius 56px
- Background: `primarySoft` (#E8F0EB)
- Content: Arabic letter in `arabicDisplay` preset, color `text`
- Behind circle: WarmGlow (animated, size: 160px, color: `accentGlow`, pulseMin: 0.06, pulseMax: 0.18) -- creates warm ambient light effect
- Entrance: Scale from 0.8 to 1.0 with `springs.gentle`, delay 100ms after card appears

**Title:** `cardHeadline` preset, color `brown`, centered, bottom margin `sm` (8px)

**Description:** `body` preset, color `textMuted`, centered, bottom margin `xxl` (32px), horizontal padding `sm` (8px)

**CTA Button:**
- Full width, `Button` component primary variant
- Label logic: "Start Lesson" (first ever) / "Continue Lesson" (in progress) / "Review Lesson" (completed)
- Haptic: `hapticTap` on press (handled by Button component)

**Entrance animation:**
- Entire card: FadeIn + translateY from 12px to 0, duration `slow` (400ms), easing `contentReveal`
- Stagger: Card enters 80ms after header

### Hero Card -- All Done State

- Same Card wrapper (elevated)
- Title: "All lessons complete!" in `cardHeadline`, color `brown`, centered
- Description: "You have completed all available lessons. Keep reviewing to strengthen your knowledge." in `body`, color `textMuted`, centered
- No CTA button, no letter circle, no phase pill

### Journey Path Section

**Section header:**
- `sectionHeader` preset (uppercase, 1.5px tracking), color `brownLight`
- Bottom margin: `xl` (24px)
- Content: "PHASE {N} -- {PHASE NAME}"

**Path layout:**
- Padding left: `xxl` (32px) for connector line space
- Padding vertical: `sm` (8px)

**Connector line:**
- Position: absolute, left 50px, top to bottom
- Style: dashed, borderLeftWidth `thick` (2px), color `border`
- No changes from current implementation

**Stagger entrance:**
- Each journey node enters with FadeIn + translateY from 8px, using `staggers.fast` (50ms delay between nodes, 300ms duration)
- First node delay: 200ms after journey section header appears

### Journey Node -- Complete State

- Circle: 40x40px, borderRadius 20px, background `primary` (#163323), shadow `card`
- Content: Checkmark SVG icon (16px, color `accent`)
- Opacity: 0.85 (slightly muted to not compete with current node)
- Label: Lesson title in `bodyMedium` 15px, color `text`; subtitle "Completed" in `bodyRegular` 12px, color `textSoft`
- Press: `springs.press` scale animation + `hapticTap`, navigates to lesson

### Journey Node -- Current State

- Circle: 44x44px (larger than others), borderRadius 22px, background `bgCard`, border `thick` (2px) `primary`, shadow `card`
- Content: Solid dot 12x12px, borderRadius 6px, background `primary`
- Subtle glow: Accent-colored ring animation -- a faint pulsing glow ring (opacity 0.08 to 0.15, 3000ms cycle) around the circle using `accentGlow` color, size 52px, to draw the eye without being aggressive
- Opacity: 1.0 (full prominence)
- Label card: background `bgCard`, border 1px `border`, borderRadius `lg` (16px), padding `md`/`lg` vertical/horizontal, shadow `card`
- Label title: `cardHeadline` at 15px override, color `brown`
- "Up next" row: Dot 6px `accent` + text `label` preset `accent` color, gap `xs`, top margin `xs`
- Press: `springs.press` scale animation + `hapticTap`, navigates to lesson

### Journey Node -- Locked State

- Circle: 40x40px, borderRadius 20px, background `bg`, border `thick` (2px) `border`
- Content: Arabic letter preview (Amiri 18px, color `textMuted`) if teachIds exist, otherwise LockIcon (14px, color `textMuted`)
- Opacity: 0.4 (clearly dimmed)
- Label: Lesson title in `bodyMedium` 15px, color `textMuted`; subtitle "Locked" in `bodyRegular` 12px, color `textMuted`
- Press: Disabled, no interaction

### Serpentine Offsets

- Pattern repeating every 6 nodes: [4, 16, 8, -4, -12, 0] px horizontal translateX
- Applied via `transform: [{ translateX: offset }]`
- No change from current implementation

## Interaction Contracts

### Press Feedback

| Element | Scale | Spring | Haptic |
|---------|-------|--------|--------|
| Hero card CTA button | `pressScale.normal` (0.97) | `springs.press` | `hapticTap` |
| Complete lesson node | `pressScale.subtle` (0.98) | `springs.press` | `hapticTap` |
| Current lesson node | `pressScale.subtle` (0.98) | `springs.press` | `hapticTap` |
| Locked lesson node | none (disabled) | none | none |

### Entrance Animations

All entrance animations use `easings.contentReveal` (Easing.out(cubic)) and run once on mount.

| Element | Type | Duration | Delay | Details |
|---------|------|----------|-------|---------|
| Header ("tila" text) | FadeIn | 300ms (`normal`) | 0ms | Immediate |
| Streak badge | FadeIn | 300ms (`normal`) | 200ms | After header |
| Hero card | FadeIn + translateY(12) | 400ms (`slow`) | 80ms | After header |
| Hero letter circle | Scale 0.8 to 1.0 | spring `gentle` | 100ms after card | Layered on card entrance |
| Journey section header | FadeIn | 300ms (`normal`) | 160ms | After hero card starts |
| Journey nodes | FadeIn + translateY(8) | 300ms (stagger `fast`) | 50ms between each, first at 200ms after section header | Sequential stagger |

### Scroll Behavior

- Standard ScrollView, no sticky headers
- `showsVerticalScrollIndicator: false`
- Content padding: `xl` (24px) on all sides
- Bottom padding: 96px (for tab bar clearance)
- No pull-to-refresh (offline-first, data is local)

### Navigation

| Action | Target | Transition |
|--------|--------|------------|
| Tap hero CTA / lesson node | `/lesson/[id]` | `slide_from_bottom` 400ms (already configured in _layout.tsx) |
| Tab switch (Progress tab) | `/(tabs)/progress` | Native tab transition (handled by tab navigator) |

## Accessibility Notes

- All interactive lesson nodes have `accessibilityRole="button"` (from Pressable)
- Locked nodes have `disabled={true}` and will be skipped by screen readers in interaction mode
- Hero CTA Button has `accessibilityRole="button"` (from Button component)
- Streak badge is read-only informational -- no special accessibility role needed
- Current node label "Up next" provides screen reader context for which lesson is active
- Color contrast: `text` (#163323) on `bg` (#F8F6F0) = 10.2:1 ratio (AAA); `textMuted` (#6B6760) on `bg` = 4.8:1 (AA); `accent` (#C4A464) on `bg` = 3.1:1 (used decoratively only, never for essential information without text backup)
