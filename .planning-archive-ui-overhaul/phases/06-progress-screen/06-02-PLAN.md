Create a reusable entrance animation pattern. Define 4 sections with staggered delays:
- Header + StatsRow: delay 0 (immediate)
- Phase Progress section header + panels: delay `staggers.normal.delay * 1` (80ms)
- Letter Mastery section header + grid: delay `staggers.normal.delay * 2` (160ms)

For each section, create a shared value pair (opacity starts at 0, translateY starts at 12):
```typescript
const statsOpacity = useSharedValue(0);
const statsTranslateY = useSharedValue(12);
const phasesOpacity = useSharedValue(0);
const phasesTranslateY = useSharedValue(12);
const masteryOpacity = useSharedValue(0);
const masteryTranslateY = useSharedValue(12);
```

In a `useEffect(() => { ... }, [])`, trigger all animations:
```typescript
// Stats section (immediate)
statsOpacity.value = withTiming(1, { duration: durations.normal, easing: easings.contentReveal });
statsTranslateY.value = withTiming(0, { duration: durations.normal, easing: easings.contentReveal });

// Phases section (staggered)
phasesOpacity.value = withDelay(staggers.normal.delay, withTiming(1, { duration: durations.normal, easing: easings.contentReveal }));
phasesTranslateY.value = withDelay(staggers.normal.delay, withTiming(0, { duration: durations.normal, easing: easings.contentReveal }));

// Mastery section (more stagger)
masteryOpacity.value = withDelay(staggers.normal.delay * 2, withTiming(1, { duration: durations.normal, easing: easings.contentReveal }));
masteryTranslateY.value = withDelay(staggers.normal.delay * 2, withTiming(0, { duration: durations.normal, easing: easings.contentReveal }));
```
