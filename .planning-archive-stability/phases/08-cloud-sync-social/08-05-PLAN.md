**Create `src/social/types.ts`**
```typescript
export interface FriendStreak {
  friendId: string;
  friendName: string | null;
  streakCount: number;
}

export interface InviteCode {
  code: string;
  expiresAt: string;
}

export type FriendRequestStatus = 'pending' | 'accepted' | 'rejected';

export interface SocialState {
  friends: FriendStreak[];
  pendingRequests: number;
  loading: boolean;
  inviteCode: InviteCode | null;
}

export interface SocialContextValue extends SocialState {
  generateInvite: () => Promise<InviteCode>;
  acceptInvite: (code: string) => Promise<{ error: Error | null }>;
  removeFriend: (friendId: string) => Promise<void>;
  refresh: () => Promise<void>;
}
```

**Create `src/social/friends.ts`** — Friend CRUD via Supabase (per D-09, D-10, D-11)
- `getFriendStreaks(supabase, userId)`: query `friend_streaks` view where `viewer_id = userId`, returns `FriendStreak[]`
- `sendFriendRequest(supabase, userId, friendId)`: insert into `friendships` with `status: 'pending'`
- `acceptFriendRequest(supabase, friendshipId)`: update `friendships` set `status: 'accepted'`
- `removeFriend(supabase, userId, friendId)`: delete from `friendships` where matches
- `getPendingRequestCount(supabase, userId)`: count friendships where `friend_id = userId` and `status = 'pending'`
- All functions wrap Supabase calls in try/catch, return typed results

**Create `src/social/invite.ts`** — Invite code system (per D-09)
- `generateInviteCode(supabase, userId)`: 
  - Generate 8-character alphanumeric code (use `Math.random().toString(36).substring(2, 10).toUpperCase()`)
  - Insert into `invite_codes` table with 7-day expiry
  - Return `InviteCode { code, expiresAt }`
- `resolveInviteCode(supabase, code)`:
  - Query `invite_codes` where `code = code` and `expires_at > NOW()`
  - If found, return the `user_id` of the inviter
  - If expired or not found, return null
- `shareInviteLink(code)`:
  - Uses `Share.share()` from react-native with message: `"Learn to read Quran with me on Tila! Use my invite code: ${code} — or open this link: tila://invite/${code}"`

**Create `src/social/provider.tsx`** — SocialProvider
- Context wrapping social state
- On mount (if authenticated): load friend streaks and pending request count
- Expose `generateInvite`, `acceptInvite`, `removeFriend`, `refresh`
- `acceptInvite(code)`: resolves code → gets inviter userId → sends friend request (both directions for mutual) → refreshes

**Create `src/social/hooks.ts`**
```typescript
export function useFriends(): SocialContextValue {
  const context = useContext(SocialContext);
  if (!context) throw new Error('useFriends must be used within SocialProvider');
  return context;
}
```
