**Create `src/auth/apple.ts`** — Native Apple Sign-In (per D-02)
Follow the exact pattern from 08-RESEARCH.md Code Examples "Native Apple Sign-In":
- Import `* as AppleAuthentication` from `'expo-apple-authentication'`
- Import `* as Crypto` from `'expo-crypto'`
- Import `{ supabase }` from `'./supabase'`
- `signInWithApple()` async function:
  1. Generate nonce: `const rawNonce = Crypto.randomUUID()`
  2. Hash nonce: `const hashedNonce = await Crypto.digestStringAsync(Crypto.CryptoDigestAlgorithm.SHA256, rawNonce)`
  3. Call `AppleAuthentication.signInAsync({ requestedScopes: [FULL_NAME, EMAIL], nonce: hashedNonce })`
  4. Call `supabase.auth.signInWithIdToken({ provider: 'apple', token: credential.identityToken!, nonce: rawNonce })`
  5. Capture `credential.fullName?.givenName` on first sign-in (pitfall #4: Apple only sends name once) — call `supabase.auth.updateUser({ data: { full_name: ... } })`
  6. Return `{ data, error }`
- Also export `isAppleSignInAvailable()`: wraps `AppleAuthentication.isAvailableAsync()`

**Create `src/auth/google.ts`** — Native Google Sign-In (per D-02)
Follow the exact pattern from 08-RESEARCH.md Code Examples "Native Google Sign-In":
- Import `{ GoogleSignin }` from `'@react-native-google-signin/google-signin'`
- Import `{ supabase }` from `'./supabase'`
- Call `GoogleSignin.configure({ webClientId: process.env.EXPO_PUBLIC_GOOGLE_WEB_CLIENT_ID! })` at module level
- `signInWithGoogle()` async function:
  1. `await GoogleSignin.hasPlayServices()`
  2. `const response = await GoogleSignin.signIn()`
  3. Validate `response.data?.idToken` exists (throw if not)
  4. Call `supabase.auth.signInWithIdToken({ provider: 'google', token: response.data.idToken })`
  5. Return `{ data, error }`

**Create `src/auth/email.ts`** — Email auth (per D-02)
- Import `{ supabase }` from `'./supabase'`
- `signInWithEmail(email: string, password: string)`: calls `supabase.auth.signInWithPassword({ email, password })`, returns `{ data, error }`
- `signUpWithEmail(email: string, password: string)`: calls `supabase.auth.signUp({ email, password })`, returns `{ data, error }`
- `signOut()`: calls `supabase.auth.signOut()`
- `resetPassword(email: string)`: calls `supabase.auth.resetPasswordForEmail(email)`

All functions wrap Supabase calls in try/catch, returning `{ data: null, error: new Error(message) }` on failure.
