> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship Wave 1 of the Phase 1 batch design — L2 + L3 lessons, Choose + Build renderers, lesson grid, AsyncStorage progress, two analytics events — gated by an end-of-wave Reality Check on 5–10 observed beginner sessions.

**Architecture:** Lessons are hand-compiled `LessonData` artifacts driven by the existing `LessonRunner`. New `progress-store.ts` replaces `completion-store.ts` with a richer schema (`{ completedLessonIds, lastReachedLessonId }`) keyed at `tila.progress`. Grid state derives purely from progress so lesson cells render `completed` / `current` / `unlocked` / `locked` from one read. Two new exercise renderers (Choose, Build) compose against the existing `renderExercise` dispatcher. Two new analytics events (`lesson_start`, `lesson_complete`) flow through the existing typed event map; `firstTryCorrectRate` is a derived field on `lesson_complete`.

**Tech Stack:** Expo SDK 55 / React Native 0.83, TypeScript 5.9 strict, Vitest 4.x, AsyncStorage, expo-audio. Tests run via `npx vitest run`. No new top-level dependencies introduced.

**Hard constraints (do not violate):**

- Reality Check is a hard gate. Do not author L4–L8 in this plan.
- BuildExercise has a Day-3 kill-switch (Checkpoint 3). If Build is not shippable by Day 3, drop it from L3, swap to Choose+Tap, defer Build to Wave 2.
- No mastery, monetization, cloud sync, accounts, habit/streak, social, or per-question scoring work.
- No new `Exercise` types beyond Choose and Build. No new `TeachingBlock` primitives — compose, do not create.

```ts
import AsyncStorage from "@react-native-async-storage/async-storage";

const KEY = "tila.progress";

export interface ProgressState {
  completedLessonIds: string[];
  lastReachedLessonId: string | null;
}

const INITIAL: ProgressState = {
  completedLessonIds: [],
  lastReachedLessonId: null,
};

const LESSON_SEQUENCE = [
  "lesson-01",
  "lesson-02",
  "lesson-03",
  "lesson-04",
  "lesson-05",
  "lesson-06",
  "lesson-07",
  "lesson-08",
];

async function read(): Promise<ProgressState> {
  try {
    const raw = await AsyncStorage.getItem(KEY);
    if (!raw) return { ...INITIAL };
    return JSON.parse(raw) as ProgressState;
  } catch (err) {
    if (typeof __DEV__ !== "undefined" && __DEV__) {
      // eslint-disable-next-line no-console
      console.warn("[progress-store] read failed", err);
    }
    return { ...INITIAL };
  }
}

async function write(state: ProgressState): Promise<void> {
  try {
    await AsyncStorage.setItem(KEY, JSON.stringify(state));
  } catch (err) {
    if (typeof __DEV__ !== "undefined" && __DEV__) {
      // eslint-disable-next-line no-console
      console.warn("[progress-store] write failed", err);
    }
  }
}

export const progressStore = {
  async getProgress(): Promise<ProgressState> {
    return read();
  },

  async markCompleted(lessonId: string): Promise<void> {
    const state = await read();
    if (state.completedLessonIds.includes(lessonId)) return;
    await write({
      ...state,
      completedLessonIds: [...state.completedLessonIds, lessonId],
    });
  },

  async setLastReached(lessonId: string): Promise<void> {
    const state = await read();
    await write({ ...state, lastReachedLessonId: lessonId });
  },

  async isUnlocked(lessonId: string): Promise<boolean> {
    if (lessonId === LESSON_SEQUENCE[0]) return true;
    const idx = LESSON_SEQUENCE.indexOf(lessonId);
    if (idx === -1) return false;
    const prev = LESSON_SEQUENCE[idx - 1];
    const state = await read();
    return state.completedLessonIds.includes(prev);
  },
};
```

```ts
import type { ProgressState } from "./progress-store";

export type LessonCellState = "completed" | "current" | "unlocked" | "locked";

export interface LessonCell {
  lessonId: string;
  state: LessonCellState;
}

/**
 * Pure derivation: maps a sequence of lesson ids and a progress snapshot to
 * per-cell render states. The first uncompleted unlocked lesson is "current"
 * (the "you are here" cell). All cells past the first incomplete lesson are
 * "locked" until prior cells complete.
 */
export function deriveLessonGridState(
  lessonIds: string[],
  progress: ProgressState
): LessonCell[] {
  const completed = new Set(progress.completedLessonIds);
  let foundCurrent = false;
  return lessonIds.map((id, idx) => {
    if (completed.has(id)) {
      return { lessonId: id, state: "completed" as const };
    }
    const prevId = idx > 0 ? lessonIds[idx - 1] : null;
    const prevDone = prevId === null || completed.has(prevId);
    if (prevDone && !foundCurrent) {
      foundCurrent = true;
      return { lessonId: id, state: "current" as const };
    }
    return {
      lessonId: id,
      state: prevDone ? ("unlocked" as const) : ("locked" as const),
    };
  });
}
```

```tsx
import React, { useEffect, useState } from "react";
import {
  View,
  Text,
  Pressable,
  StyleSheet,
  AccessibilityInfo,
} from "react-native";
import Animated, {
  useSharedValue,
  useAnimatedStyle,
  withRepeat,
  withSequence,
  withTiming,
  cancelAnimation,
} from "react-native-reanimated";
import { useColors } from "../../design/theme";
import { typography, spacing, radii } from "../../design/tokens";
import type { LessonCell } from "../../curriculum/runtime/grid-state";

interface LessonGridProps {
  cells: LessonCell[];
  titles: Record<string, string>;
  onPress: (lessonId: string) => void;
}

const CELL_LABEL = (cell: LessonCell, title: string): string => {
  const status =
    cell.state === "completed"
      ? "completed"
      : cell.state === "current"
        ? "next up"
        : cell.state === "locked"
          ? "locked"
          : "available";
  return `${title}, ${status}`;
};

export function LessonGrid({ cells, titles, onPress }: LessonGridProps) {
  const colors = useColors();
  const [reduceMotion, setReduceMotion] = useState(false);
  const glow = useSharedValue(0);

  useEffect(() => {
    let alive = true;
    AccessibilityInfo.isReduceMotionEnabled().then((value) => {
      if (alive) setReduceMotion(value);
    });
    const sub = AccessibilityInfo.addEventListener(
      "reduceMotionChanged",
      setReduceMotion
    );
    return () => {
      alive = false;
      sub.remove();
    };
  }, []);

  useEffect(() => {
    if (reduceMotion) {
      cancelAnimation(glow);
      glow.value = 0;
      return;
    }
    glow.value = withRepeat(
      withSequence(
        withTiming(1, { duration: 1400 }),
        withTiming(0, { duration: 1400 })
      ),
      -1,
      false
    );
    return () => cancelAnimation(glow);
  }, [reduceMotion, glow]);

  const glowStyle = useAnimatedStyle(() => ({
    opacity: 0.4 + glow.value * 0.5,
  }));

  return (
    <View style={styles.column}>
      {cells.map((cell) => {
        const title = titles[cell.lessonId] ?? cell.lessonId;
        const interactive =
          cell.state === "completed" ||
          cell.state === "current" ||
          cell.state === "unlocked";
        const isCurrent = cell.state === "current";
        return (
          <Pressable
            key={cell.lessonId}
            onPress={interactive ? () => onPress(cell.lessonId) : undefined}
            disabled={!interactive}
            accessibilityRole={interactive ? "button" : "text"}
            accessibilityLabel={CELL_LABEL(cell, title)}
            style={[
              styles.cell,
              {
                backgroundColor: colors.surface,
                borderColor:
                  cell.state === "current"
                    ? colors.accent
                    : colors.border,
                opacity: cell.state === "locked" ? 0.45 : 1,
              },
            ]}
          >
            {isCurrent && !reduceMotion && (
              <Animated.View
                pointerEvents="none"
                style={[
                  StyleSheet.absoluteFill,
                  {
                    borderRadius: radii.lg,
                    borderWidth: 2,
                    borderColor: colors.accent,
                  },
                  glowStyle,
                ]}
              />
            )}
            <View style={styles.row}>
              <Text style={[styles.title, { color: colors.text }]}>
                {title}
              </Text>
              <Text style={[styles.badge, { color: colors.textSubtle }]}>
                {cell.state === "completed"
                  ? "Done"
                  : cell.state === "current"
                    ? "Start"
                    : cell.state === "locked"
                      ? "Locked"
                      : "Open"}
              </Text>
            </View>
          </Pressable>
        );
      })}
    </View>
  );
}

const styles = StyleSheet.create({
  column: { gap: spacing.md, paddingHorizontal: spacing.md },
  cell: {
    minHeight: 72,
    borderRadius: radii.lg,
    borderWidth: 1,
    paddingHorizontal: spacing.md,
    paddingVertical: spacing.md,
    justifyContent: "center",
    overflow: "hidden",
  },
  row: { flexDirection: "row", justifyContent: "space-between", alignItems: "center" },
  title: { ...typography.titleSmall },
  badge: { ...typography.caption },
});
```

```tsx
import React, { useCallback, useEffect, useRef, useState } from "react";
import { View, Text, StyleSheet } from "react-native";
import { HearButton } from "../../../design/components";
import { QuizOption } from "../../../design/components";
import { useColors } from "../../../design/theme";
import { typography, spacing } from "../../../design/tokens";
import type {
  ChooseExercise as ChooseExerciseType,
  EntityKey,
} from "../../types";
import type { EntityAttempt, ScreenOutcome } from "../../runtime/outcome";

interface Props {
  exercise: ChooseExerciseType;
  screenId: string;
  retryMode?: "until-correct" | "one-shot";
  advance: (outcome?: ScreenOutcome) => void;
  reportAttempt: (event: EntityAttempt) => void;
  onPlayAudio?: (path: string) => void;
}

export function ChooseExerciseRenderer({
  exercise,
  screenId,
  retryMode = "until-correct",
  advance,
  reportAttempt,
  onPlayAudio,
}: Props) {
  const colors = useColors();
  const [optionStates, setOptionStates] = useState<Record<string, "idle" | "correct" | "wrong">>({});
  const [attempts, setAttempts] = useState(0);
  const advanceTimer = useRef<ReturnType<typeof setTimeout> | null>(null);

  useEffect(() => () => {
    if (advanceTimer.current) clearTimeout(advanceTimer.current);
  }, []);

  const handlePlayPrompt = useCallback(() => {
    if (exercise.audioPrompt && onPlayAudio) onPlayAudio(exercise.audioPrompt);
  }, [exercise.audioPrompt, onPlayAudio]);

  const handleSelect = useCallback(
    (entityKey: EntityKey, correct: boolean) => {
      const nextAttempts = attempts + 1;
      setAttempts(nextAttempts);
      reportAttempt({
        screenId,
        target: exercise.target,
        selected: entityKey,
        correct,
      });

      if (correct) {
        setOptionStates((prev) => ({ ...prev, [entityKey]: "correct" }));
        advanceTimer.current = setTimeout(() => {
          advance({
            screenId,
            scored: true,
            countsAsDecoding: false,
            attempts: nextAttempts,
            passed: true,
          });
        }, 600);
        return;
      }

      setOptionStates((prev) => ({ ...prev, [entityKey]: "wrong" }));
      if (retryMode === "one-shot") {
        advanceTimer.current = setTimeout(() => {
          advance({
            screenId,
            scored: true,
            countsAsDecoding: false,
            attempts: nextAttempts,
            passed: false,
          });
        }, 800);
      } else {
        advanceTimer.current = setTimeout(() => {
          setOptionStates((prev) => ({ ...prev, [entityKey]: "idle" }));
        }, 600);
      }
    },
    [advance, attempts, exercise.target, reportAttempt, retryMode, screenId]
  );

  return (
    <View style={styles.container}>
      {exercise.audioPrompt && (
        <HearButton onPress={handlePlayPrompt} disabled={!exercise.audioPrompt} />
      )}
      <Text style={[styles.prompt, { color: colors.text }]}>{exercise.prompt}</Text>
      <View style={styles.options}>
        {exercise.options.map((opt) => (
          <QuizOption
            key={opt.entityKey}
            display={opt.display}
            state={optionStates[opt.entityKey] ?? "idle"}
            onPress={() => handleSelect(opt.entityKey, opt.correct)}
            disabled={optionStates[opt.entityKey] === "correct"}
          />
        ))}
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: { flex: 1, alignItems: "center", padding: spacing.md, gap: spacing.lg },
  prompt: { ...typography.bodyLarge, textAlign: "center" },
  options: { width: "100%", gap: spacing.md },
});
```

> **Note:** if `QuizOption` or `HearButton` props differ from what is shown above (e.g., `state` may be a different name or shape), match the existing component API. Read `src/design/components/QuizOption.tsx` and `src/design/components/HearButton.tsx` first.

- [ ] **Step 4: Implement the renderer**

Create `src/curriculum/ui/exercises/BuildExercise.tsx`. Interaction: tap tiles in sequence; each tap appends to a "construction" row at the top; check-on-submit (no per-tap error). Provide an "undo last" button. On submit, compare assembled sequence to `correctSequence`. On match, fire correct outcome and advance. On mismatch in `until-correct` mode, clear the construction row and let learner retry.

```tsx
import React, { useCallback, useEffect, useRef, useState } from "react";
import { View, Text, Pressable, StyleSheet } from "react-native";
import { useColors } from "../../../design/theme";
import { typography, spacing, radii } from "../../../design/tokens";
import type {
  BuildExercise as BuildExerciseType,
  EntityKey,
} from "../../types";
import type { EntityAttempt, ScreenOutcome } from "../../runtime/outcome";

interface Props {
  exercise: BuildExerciseType;
  screenId: string;
  retryMode?: "until-correct" | "one-shot";
  advance: (outcome?: ScreenOutcome) => void;
  reportAttempt: (event: EntityAttempt) => void;
  onPlayAudio?: (path: string) => void;
}

export function BuildExerciseRenderer({
  exercise,
  screenId,
  retryMode = "until-correct",
  advance,
  reportAttempt,
}: Props) {
  const colors = useColors();
  const [assembled, setAssembled] = useState<EntityKey[]>([]);
  const [attempts, setAttempts] = useState(0);
  const [feedback, setFeedback] = useState<"idle" | "correct" | "wrong">("idle");
  const advanceTimer = useRef<ReturnType<typeof setTimeout> | null>(null);

  useEffect(() => () => {
    if (advanceTimer.current) clearTimeout(advanceTimer.current);
  }, []);

  const sequenceMatches = useCallback((seq: EntityKey[]) => {
    if (seq.length !== exercise.correctSequence.length) return false;
    return seq.every((k, i) => k === exercise.correctSequence[i]);
  }, [exercise.correctSequence]);

  const handleTile = useCallback((entityKey: EntityKey) => {
    if (feedback !== "idle") return;
    setAssembled((prev) => [...prev, entityKey]);
  }, [feedback]);

  const handleUndo = useCallback(() => {
    if (feedback !== "idle") return;
    setAssembled((prev) => prev.slice(0, -1));
  }, [feedback]);

  const handleSubmit = useCallback(() => {
    if (feedback !== "idle") return;
    const correct = sequenceMatches(assembled);
    const nextAttempts = attempts + 1;
    setAttempts(nextAttempts);
    reportAttempt({
      screenId,
      target: exercise.target,
      selected: assembled.join("+") as EntityKey,
      correct,
    });
    setFeedback(correct ? "correct" : "wrong");

    if (correct) {
      advanceTimer.current = setTimeout(() => {
        advance({
          screenId,
          scored: true,
          countsAsDecoding: true,
          attempts: nextAttempts,
          passed: true,
        });
      }, 700);
      return;
    }

    if (retryMode === "one-shot") {
      advanceTimer.current = setTimeout(() => {
        advance({
          screenId,
          scored: true,
          countsAsDecoding: true,
          attempts: nextAttempts,
          passed: false,
        });
      }, 800);
    } else {
      advanceTimer.current = setTimeout(() => {
        setAssembled([]);
        setFeedback("idle");
      }, 700);
    }
  }, [advance, assembled, attempts, exercise.target, feedback, reportAttempt, retryMode, screenId, sequenceMatches]);

  return (
    <View style={styles.container}>
      <Text style={[styles.prompt, { color: colors.text }]}>{exercise.prompt}</Text>

      <View style={[styles.assembledRow, { borderColor: colors.border }]}>
        {assembled.length === 0 ? (
          <Text style={[styles.placeholder, { color: colors.textSubtle }]}>
            Tap tiles in order
          </Text>
        ) : (
          assembled.map((k, i) => {
            const tile = exercise.tiles.find((t) => t.entityKey === k);
            return (
              <Text
                key={`${k}-${i}`}
                style={[styles.assembledGlyph, { color: colors.text }]}
              >
                {tile?.display ?? "?"}
              </Text>
            );
          })
        )}
      </View>

      <View style={styles.tiles}>
        {exercise.tiles.map((tile) => (
          <Pressable
            key={tile.entityKey}
            onPress={() => handleTile(tile.entityKey)}
            style={[styles.tile, { borderColor: colors.border, backgroundColor: colors.surface }]}
          >
            <Text style={[styles.tileGlyph, { color: colors.text }]}>{tile.display}</Text>
          </Pressable>
        ))}
      </View>

      <View style={styles.actions}>
        <Pressable onPress={handleUndo} disabled={assembled.length === 0} style={styles.actionBtn}>
          <Text style={{ color: colors.textSubtle }}>Undo</Text>
        </Pressable>
        <Pressable
          onPress={handleSubmit}
          disabled={assembled.length === 0 || feedback !== "idle"}
          style={[styles.actionBtn, { backgroundColor: colors.accent, borderRadius: radii.md }]}
        >
          <Text style={{ color: colors.surface }}>Check</Text>
        </Pressable>
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: { flex: 1, padding: spacing.md, gap: spacing.lg },
  prompt: { ...typography.bodyLarge, textAlign: "center" },
  assembledRow: {
    minHeight: 72,
    borderWidth: 1,
    borderRadius: radii.md,
    flexDirection: "row",
    alignItems: "center",
    justifyContent: "center",
    gap: spacing.sm,
    padding: spacing.md,
  },
  assembledGlyph: { fontSize: 48 },
  placeholder: { ...typography.body },
  tiles: { flexDirection: "row", flexWrap: "wrap", gap: spacing.md, justifyContent: "center" },
  tile: {
    width: 72,
    height: 72,
    borderWidth: 1,
    borderRadius: radii.md,
    alignItems: "center",
    justifyContent: "center",
  },
  tileGlyph: { fontSize: 36 },
  actions: { flexDirection: "row", justifyContent: "space-between", gap: spacing.md },
  actionBtn: { padding: spacing.md },
});
```

> **Note:** if any token name above isn't in `src/design/tokens.ts`, substitute the closest existing token. Match `QuizOption` styling conventions where possible.

## Task 25: Reality Check Observation Kit

**Files:**
- Create: `docs/superpowers/notes/2026-04-28-reality-check-kit.md`

**Goal:** Produce the operational kit needed to run 5–10 observed beginner sessions: tester recruitment list, pre-session script, post-session question protocol, decision rule, observation log template.

- [ ] **Step 1: Author the kit**

Create `docs/superpowers/notes/2026-04-28-reality-check-kit.md`:

```markdown
# Wave 1 Reality Check — Observation Kit

## Cohort target

5–10 testers who:
- Have NO prior Arabic reading experience
- Are not engineers building learning apps
- Are willing to do a 15–25 minute observed session (in person or screen-share)

## Pre-session script (read aloud)

"Thanks for helping. I'm going to ask you to try a new app for ~20 minutes. I won't help during the session — I want to see what's confusing without being there to explain. Speak your thoughts aloud as you go: what you see, what you're trying to do, what feels stuck. Afterward I'll ask three questions. Sound good?"

## During the session

Observe silently. Take notes on:
- Where they hesitate (which screen, which exercise)
- What they say aloud
- Any retry loops (note which screen + how many attempts)
- Whether they laugh, smile, frown, or look frustrated

Do NOT explain anything. If they ask "what does this mean?", say "what do you think it means?" and write the answer down.

## Post-session questions (ask in order)

1. **What was confusing?**
2. **What felt easy?**
3. **What did you think this was teaching?** (Most important — tells you whether the lesson's intent landed.)

## Decision rule (apply after all sessions complete)

Failure criteria — treat as "fail" for this tester if they:
- Confuse letter name vs sound (e.g., say "ba" for the letter ب in isolation, not just for the syllable بَ)
- Do not internalize fatha = "a"
- Cannot read بَ consistently across L2 and L3

Decision:
- **≥ 30% of testers fail at least one criterion → BLOCK Wave 2.** Fix L1–L3 first; re-run Reality Check on a fresh cohort.
- **< 30% fail BUT confusion patterns repeat across multiple testers → FIX before scaling.** Pattern repetition = signal, not noise.
- **Isolated confusion (one tester, unique issue) → LOG, do not block.**

## Observation log template (one per session)

```
Tester: <pseudonym>
Date: <date>
Duration: <mm:ss>
L1 first-try-correct-rate: <number>
L2 first-try-correct-rate: <number>
L3 first-try-correct-rate: <number>

Hesitations:
- <screen> — <what happened>

Retry loops:
- <screen> — <attempts>

Aloud quotes:
- "<quote>"

Post-session:
1. Confusing: <answer>
2. Easy: <answer>
3. Thought it was teaching: <answer>

Failure criteria hit: [name vs sound] [fatha = a] [read بَ]

Notes:
- <free-form>
```

## Scale-up green lights (look for these in observations)

- Move faster through L2 than L1
- Predict answers before tapping
- Spontaneously generalize ("oh it's always 'a' with that line")
```
